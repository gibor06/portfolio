## Giới thiệu

Đây là một website portfolio cá nhân được xây dựng theo phong cách hiện đại, tập trung vào:

- Thể hiện thương hiệu cá nhân rõ ràng
- Trình bày kỹ năng và định hướng nghề nghiệp
- Giới thiệu case study và dự án thực tế
- Cung cấp minh chứng thành tích học thuật
- Hỗ trợ recruiter liên hệ nhanh qua form/email/lịch hẹn

Website phù hợp để sử dụng như một hồ sơ online cho vị trí:

- Intern Software Engineer
- Fresher Developer
- AI / Data / Web-related roles

## Cách chạy project trên máy

Vì đây là website tĩnh, bạn có thể chạy rất đơn giản:

### Cách 1: Mở trực tiếp
Chỉ cần mở file `index.html` bằng trình duyệt.

### Cách 2: Chạy bằng Live Server
Khuyến khích dùng VS Code + Live Server để trải nghiệm đầy đủ hơn.

1. Clone repo:

```bash
git clone https://github.com/gibor06/portfolio.git
```

2. Mở thư mục project trong VS Code

3. Chạy `index.html` bằng Live Server

## Tuỳ chỉnh nội dung

Bạn có thể chỉnh sửa nội dung chính tại:

- `index.html`: nội dung website, section, liên kết, thông tin cá nhân
- `style.css`: giao diện, màu sắc, layout, animation
- `script.js`: hiệu ứng tương tác, timeline, gallery, form submit

## Cấu hình form liên hệ

Form liên hệ hiện hỗ trợ gửi dữ liệu qua **Formspree**.

Trong `script.js`, hãy thay endpoint mẫu bằng endpoint thật của bạn:

```js
const FORMSPREE_ENDPOINT_DEFAULT = "https://formspree.io/f/your_form_id";
```

Sau đó đổi `your_form_id` thành form endpoint thực tế để form hoạt động.
